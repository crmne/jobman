
This directory contains a few files:

def_addition.py         - example 'experiment' function
addition_driver.py      - an example of how to use jobman.mydriver to manage experiments.

createview_addition.py  - manual view creation
insert_addition.py      - managing experiments without using jobman.mydriver


These files are meant to be cut-and-pasted into your own account and modified
to run your own experiments.
