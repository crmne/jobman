
These files are meant to house experiment-type specific analysis/management routines.

The pkl.py file will be imported when the analysis runner is called with an address
that starts with 'pkl://'.

The dd.py file will be imported when the analysis runner is called with an address
that starts with 'dd://'.

The pg.py file will be imported when the analysis runner is called with an address
that starts with 'postgres://'.
